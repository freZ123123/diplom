GameStore — Интернет-магазин компьютерных игр
Web приложение для продажи цифровых копий компьютерных игр, ключей активации и внутриигровых предметов

О проекте
GameStore — специализированная платформа для покупки и продажи лицензионных компьютерных игр. В отличие от Steam и Epic Games Store, предоставляет упрощенную систему лицензирования, возможность перепродажи цифровых ключей и интеграцию с популярными платежными системами СНГ.

Основные функции
🎮 Каталог игр с фильтрацией по жанрам, цене и рейтингу

🛒 Корзина и оформление заказа

🔐 Личный кабинет с историей покупок

💳 Интеграция с платежными системами (Карты, SBP, ЮMoney)

⭐ Система отзывов и рейтингов

🎁 Промокоды и сезонные скидки
